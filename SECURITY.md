# Security Policy

## Supported Project

This repository contains the website for Summit Structure Supplies.

## Reporting a Vulnerability

If you find a security issue, do not open a public GitHub issue.

Please report it privately to the repository owner with:

- the affected page or file;
- steps to reproduce the issue;
- the possible impact;
- screenshots or logs if available.

## Sensitive Information

Do not commit or share:

- Supabase secret keys;
- service role keys;
- database passwords;
- private keys;
- customer order exports;
- database backups.

## Security Practices

This project uses:

- Supabase Row Level Security;
- administrator MFA;
- restricted public signups;
- database validation constraints;
- pinned Supabase JavaScript dependency;
- GitHub security checks.
